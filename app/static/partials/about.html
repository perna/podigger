Sobre o Podigger

Este é um "projeto de um homem só" (por enquanto) feito por Anderson Meira, o Perna do Toscochanchada Podcast.

Com a motivação inicial de aprender novas tecnologias, resolvi desenvolver uma ferramenta como prova de conceito, que tornasse simples a busca de podcasts por assuntos, palavras chaves, etc.

No desenvolvimento percebi que esta prova de conceito pode realmente ser útil para ouvintes e podcasters, então resolvi publicar de fato esta versão.

Na criação do Podigger utilizei as seguintes tecnologias:

  • Linguagem Python
  • Microframework Flask
  • PostgreSQL com Full Text Search
  • Celery e Redis para gestão de fila
  • Sendgrid para envio de emails
  • AngularJS e Bootsrap para as interfaces.

Para que eu possa melhorar a ferramenta, além das funcionalidades que ainda estão por vir, o feedback dos usuários é importante. Veja como enviar o seu feedback na página de contato.
